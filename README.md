# case-banking-Netology

В файле Netology.ipynb находится подробное решение и обьяснение шагов выполнения данного кейса.
Данный кейс был решен в ходе прохожднеия практики цифровой кафедры НИУ "МИЭТ" "Анализ данных с применением машшинного обучения".
При решении были использованы библиотеки matplotlib и seaborn для визуализации данных.
Для работы с данными были использованы numpy и pandas.
Для обучения моделей и подготовки данных: Sklearn.
