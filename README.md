# case-banking-Netology

<p>В файле Netology.ipynb находится подробное решение и обьяснение шагов выполнения данного кейса.</p>
<p>Данный кейс был решен в ходе прохожднеия практики цифровой кафедры НИУ "МИЭТ" "Анализ данных с применением машшинного обучения".</p>
<p>При решении были использованы библиотеки matplotlib и seaborn для визуализации данных.</p>
<p>Для работы с данными были использованы numpy и pandas.</p>
<p>Для обучения моделей и подготовки данных: Sklearn, Catboost.</p>
<p>Для подбора гиперпараметров: Optuna и Sklearn</p>
