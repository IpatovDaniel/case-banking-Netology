# case-banking-Netology

<p>В файле Netology.ipynb находится подробное решение и объяснение шагов выполнения кейса прогноза оттока клиентов банка на основе транзакционной активности.</p>
<p>Данный кейс был решен в процессе прохождения практики цифровой кафедры НИУ "МИЭТ" "Анализ данных с применением машинного обучения".</p>
<p>При решении были использованы библиотеки matplotlib и seaborn для визуализации данных.</p>
<p>Для работы с данными были использованы numpy и pandas.</p>
<p>Для обучения моделей и подготовки данных: Sklearn, Catboost.</p>
<p>Для подбора гиперпараметров: Optuna и Sklearn</p>
